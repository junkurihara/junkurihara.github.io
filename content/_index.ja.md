## ソフトウェア開発プロジェクト

主にOSSプロジェクトです。

- **Mutualized Oblivious DNS (μODNS)**: Anonymized DNS project — [Project page](/dns)
- **rpxy**: A simple and ultrafast reverse-proxy for multiple domain names with TLS termination, written in pure Rust — [GitHub](https://github.com/junkurihara/rust-rpxy) / [https://rpxy.io](https://rpxy.io)
- **rpxy-l4**: A reverse proxy for the layer-4 (TCP+UDP) with protocol multiplexer, written in Rust — [GitHub](https://github.com/junkurihara/rust-rpxy-l4)
- **rust-gd**: A Rust implementation of Generalized Deduplication — [GitHub](https://github.com/junkurihara/rust-gd)
- **jscu**: A Universal Cryptographic Library for JavaScript — [GitHub](https://github.com/junkurihara/jscu)
- [GitHub](https://github.com/junkurihara) も参照してください。
